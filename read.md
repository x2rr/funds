# 基金对比功能实现与构建记录

## 日期
2026-04-16

## 功能概述
为自选基金助手项目新增了基金对比功能，支持多支基金的收益率对比分析。

## 新增功能

### 1. 工具栏对比按钮
- 在主界面工具栏添加了"基金对比"按钮
- 位于"行情中心"按钮旁边

### 2. 基金对比选择页面 (fundCompareSelect.vue)
- 展示自选基金列表供选择
- 支持搜索并添加不在自选列表中的基金
- 限制最多选择3支基金进行对比
- 显示已选择的基金列表，支持移除

### 3. 基金对比结果页面 (fundCompareResult.vue)
- **对比周期**：支持近一周、近一月、近三月、近一年、近三年
  - 近一周实现：从月数据中截取最近7天的数据
- **对比概览表格**：
  - 成立时间、基金类型、最新净值、基金规模、基金公司、基金经理
  - 各周期收益率（近一月、三月、六月、一年、三年）
  - 当前选择周期的收益率（高亮显示，正数红色，负数绿色）
- **收益走势对比**：使用ECharts绘制折线图
- **基金成立时间处理**：
  - 自动检测数据完整性
  - 显示黄色警告图标和"(部分)"标记
  - 图表注记说明数据不完整情况

### 4. 近一周数据实现
由于API不直接支持周数据，通过以下方式实现：
- 使用月数据的API参数（RANGE=y）
- 从返回的月数据中截取最近7天的数据
- 计算这7天的收益率

## 新增/修改的文件

- `src/common/fundCompareSelect.vue` - 基金对比选择组件

## 构建过程

### 环境
- Windows系统 + WSL (Windows Subsystem for Linux)
- Node.js: v12.22.9
- npm: 8.5.1

### 步骤

#### 1. 安装依赖
```bash
cd /mnt/d/fund_helper/funds
npm install --legacy-peer-deps
```

#### 2. 构建项目
```bash
cd /mnt/d/fund_helper/funds
NODE_ENV=production ./node_modules/.bin/webpack
```
或者使用 npm scripts：
```bash
npm run build
```

#### 3. 生成压缩包
```bash
npm run build-zip
```

### 构建输出
- `dist/` 目录：包含 Chrome 扩展所需的所有文件
  - `background.js`
  - `manifest.json`
  - `popup/` 目录
  - `options/` 目录
  - `fonts/` 目录
  - `icons/` 目录

- `dist-zip/` 目录：包含生成的压缩包
  - `choose-funds-v2.5.2.zip` (865,031 字节)

## 验证方法

### 调试模式
1. 执行 `npm run watch:dev` 生成 dist 文件夹
2. 打开 Chrome 浏览器，进入扩展管理页面
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择生成的 `dist` 目录

1. 执行 `npm run build` 生成 dist 文件夹
2. 执行 `npm run build-zip` 生成压缩包
3. 在 Chrome 扩展管理页面，直接拖放 zip 文件进行安装
4. 或者解压后通过"加载已解压的扩展程序"安装
